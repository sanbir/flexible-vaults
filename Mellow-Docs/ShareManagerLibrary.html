ShareManagerLibrary

This library helps pack multiple boolean flags and lockup durations into a compact
uint256
bitmask. It enables efficient storage and quick access to share manager configuration in vault systems.
Designed for the
ShareManager
component to control:
Whether minting, burning, or transfers are paused
Whether deposit/transfer whitelists are active
How long global or user-specific lockups last
All data is packed into a single
uint256
using bit-level encoding for optimal storage and gas efficiency.
