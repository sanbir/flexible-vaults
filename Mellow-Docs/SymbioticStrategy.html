SymbioticStrategy

Overview

The
SymbioticStrategy
contract provides an out-of-scope example implementation of the Strategy contract for interacting with a Symbiotic vault via a
subvault
using a
CallModule
interface. It allows depositing assets into, withdrawing from, and claiming withdrawals from a Symbiotic Vault and claiming rewards from Symbiotic farm by executing safe, externally verified low-level calls through the vault’s module system.
Note: This contract is not intended for production use and is considered out of scope. It serves solely as a reference implementation demonstrating how strategy contracts can interact with Subvaults via CallModule.call.
